Tamizaje emocional breve

DASS-21 digital.

Escala breve para estimar niveles recientes de depresión, ansiedad y estrés durante la última semana. El reporte automatiza el puntaje por subescala, la equivalencia de severidad y una síntesis clínica orientativa.

21 ítems con respuesta de 0 a 3 según frecuencia o intensidad.
Subescalas independientes: depresión, ansiedad y estrés.
Puntaje final ajustado multiplicando por 2 para equivalencia DASS completa.
Útil como tamizaje. No reemplaza evaluación diagnóstica profesional.
